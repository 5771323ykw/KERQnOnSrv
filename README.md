# Python学生宿舍管理系统

## 核心功能
- 宿舍分配
- 宿舍保修
- 宿舍查询
- 学生信息管理
- 宿舍信息管理
- 通知管理

## 适用场景
- 学生宿舍管理部门
- 高校宿舍管理
- 宿舍管理服务提供者

## 技术特点
- 使用Python Flask框架
- 前端采用HTML、CSS、JavaScript
- 后端使用MySQL数据库
- 支持Pycharm、navicat工具
- 环境要求：Python3.0以上、MySQL8.0
